# DOE — Effect of Soil Nutrients (N, P, K) on Soil pH

A **2³ Full Factorial Design of Experiments (DoE)** analyzing how Nitrogen (N), Phosphorus (P), and Potassium (K) affect soil pH, using the Crop Recommendation Dataset.

---

## Objective

To determine which soil nutrients — individually and in combination — significantly affect soil pH using a Full Factorial experimental design.

---

## 🔬 Experiment Design

| Element | Detail |
|---|---|
| **Design Type** | 2³ Full Factorial |
| **Factors** | Nitrogen (N), Phosphorus (P), Potassium (K) |
| **Levels** | 2 — Low (−1) and High (+1) based on median |
| **Response** | Soil pH |
| **Total Runs** | 8 treatment combinations |
